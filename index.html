Prazer, Lucas

Estudando para ser desenvolvedor

Sou um profissional apaixonado por Desenvolvimento de Software. Durante meus 4 anos e meio na Sofit - Gestão de Frotas, tive a oportunidade de desempenhar uma variedade de funções desafiadoras. Iniciei minha jornada como Analista em BPO, onde gerenciei de forma terceirizada os custos de frotas de nossos clientes, importando dados para o sistema e criando relatórios detalhados. Minha habilidade em análise de dados me permitiu desenvolver dashboards personalizados, fornecendo informações críticas para auxiliar nossos clientes a tomarem melhores decisões.

No meu último ano na empresa, iniciei uma transição para o setor de Desenvolvimento de Software, onde assumi a responsabilidade pelo setor de integração. Nessa função, mantive e desenvolvi novas integrações entre diversos sistemas utilizados pela empresa, utilizando ferramentas No-code como Make e Jitterbit. Além disso, realizei automações de processos, utilizando Node.js para manipular planilhas em Excel e executando APIs para importação de dados no sistema. Minha jornada na Sofit me proporcionou uma base sólida de habilidades técnicas e uma mentalidade orientada para soluções, pronta para enfrentar novos desafios no mundo do desenvolvimento de software.
